Requirements
============

  + Node.js
  + Express 
  + Primus.io
  + ws
  + cors

Usage
=====

Install Dependencies:

    $ npm install

Start Server:

    $ npm start  


RESULTS PAGE 
============

- cookie type name
- total score
- category breakdown
- sorted


COOKIE REGISTRATION
===================

- ability to enter cookie type
- propagates to voting pages


VOTING PAGE
===========

- displays all cookie names in an acordian
- expanding displays the voting categories
- entered votes post to results page
